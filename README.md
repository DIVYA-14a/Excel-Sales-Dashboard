An interactive **Sales Insights Dashboard** built using Microsoft Excel to analyze sales performance, customer profitability, and regional sales distribution.

---

## 📌 Project Overview
This project transforms raw sales data into **clear and actionable insights** using Excel data visualization techniques.  
The dashboard helps identify **top-performing categories, profitable customers, and sales trends** to support better business decisions.

---

## 🛠 Tools & Technologies
- 📗 Microsoft Excel  
- 📊 Pivot Tables  
- 📉 Pivot Charts  
- 📈 Data Visualization  
- 🧹 Data Cleaning  

---

## ✨ Key Features
- 📦 **Sales by Category** – Identify the highest revenue product categories  
- 📈 **Profit Gained Over Time** – Track yearly profit trends  
- 🗺 **Sales by State** – Visualize geographic sales distribution  
- 👥 **Top 5 Customers Making Profits** – Identify valuable customers  
- 📅 **Monthly Sales Trend** – Analyze monthly sales patterns  

---

## 💡 Insights from the Dashboard
- Technology products generate the highest profits.  
- A few customers contribute significantly to total profit.  
- Sales peak during certain months of the year.  
- Some states contribute more to overall sales performance.
